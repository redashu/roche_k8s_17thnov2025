### Helm Commands

```bash
# Add Helm repositories
helm repo add bitnami https://charts.bitnami.com/bitnami
helm repo add ashu-k8s-dashboard https://kubernetes.github.io/dashboard/

# List repositories
helm repo ls

# Search for charts
helm search hub mysql
helm search repo nginx
helm search repo mysql

# Install a chart
helm install my-webapp ashu/nginx
```
